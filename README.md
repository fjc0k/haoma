# 好码

使用 **[EditorConfig](https://editorconfig.org/)** + **[ESLint](https://eslint.org/)** + **[Prettier](https://prettier.io/)**，助你写出漂亮的前端代码。


## 安装

```bash
# npm
npm i haoma -D

# or yarn
yarn add haoma -D
```


## 使用

使用以下命令初始化 `.editorconfig`、`.eslintrc.js`、`.eslintignore`、`.prettierrc.js`、`.prettierignore`：

```bash
# npm
npx haoma init

# or yarn
yarn haoma init
```
